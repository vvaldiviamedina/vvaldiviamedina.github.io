A tour of what Quarto can do

This page is a grab-bag of Quarto features, rendered right in your site. Open the .qmd source alongside this page to see how each piece is written.

Code blocks

Plain fenced code blocks get syntax highlighting for free, with optional line numbers:

def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("world"))

You can also fold long code blocks so they don’t dominate the page — click to expand:

import math

def area_of_circle(radius: float) -> float:
    return math.pi * radius ** 2

for r in range(1, 5):
    print(f"radius={r} -> area={area_of_circle(r):.2f}")

Live, interactive demo (Observable JS)

Quarto ships with Observable JS built in — no Python/R kernel required, it runs straight in the browser. Drag the slider:

viewof n = Inputs.range([1, 50], {value: 10, step: 1, label: "n"})
md`**${n} squared is ${n * n}.**`
Plot.plot({
  marks: [
    Plot.barY(
      Array.from({length: n}, (_, i) => ({x: i + 1, y: (i + 1) ** 2})),
      {x: "x", y: "y", fill: "steelblue"}
    )
  ],
  y: {label: "value²"},
  x: {label: "value"}
})
